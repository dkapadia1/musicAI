# musicAI
Use of facebook's encodec to create embeddings and find matches without bias for songs


Run demoapp.py with the model downloaded to access a gradio page showing a simple UI. 

Insert two audio files, or one audio file and a folder to compare two. The model will then compress, quantize and compare songs purely based on sound to give a similarity score. 


Using MusicGen's Encoder, Encodec to compress and compare.
